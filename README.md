# GD Textures

This repository holds the resources of the Steam version of Geometry Dash from version 2.113 up until the latest version of the game, as well as the place to host the UHD texture packs for my Geode mod "High Graphics on Mobile".

All assets in this repository are owned by Robtop Games.

## Script usage
```
python get_high.py [input path] [output path]
```
input path: your target resources folder

output path: where you want to store the high textures
